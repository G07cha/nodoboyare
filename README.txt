## Идеи

Тут указаны наши идеи на хакатон и короткое описание каждой из них, чтобы увидеть подробное описание нажмите на название идеи

- Clever pipes
Набор устройств которые позволяют управлять водными ресурсами(питьевыми в том числе) с помощью мобильного приложения. Проще говоря гибкая система которую можно использовать для полива огорода или набора жидкости в разные емкости(чайник, кострюли на плите и т.д)
